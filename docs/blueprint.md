# **App Name**: FinTrack Mobile

## Core Features:

- Home Dashboard: Interactive dashboard for visualizing net worth, cash flow, and budget progress.
- Manual Transaction Entry: Manual transaction entry using a batch entry screen with a list-style form.
- Asset Dashboard: Dashboard to manually track bank accounts, stocks, crypto, mutual funds, and property, and to view historical asset values.
- AI Insights: A tool to provide savings opportunity suggestions to users.
- Widgets and Notifications: Android home screen widgets for budget overview and quick transaction entry.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and security.
- Background color: Light gray (#F0F2F5), provides a clean, neutral backdrop.
- Accent color: Purple (#7E57C2), used for highlighting key interactions and elements.
- Roboto for headings, Google Sans for body text to provide clarity.
- Grid-based layout with rounded 2xl cards for a modern, organized look.
- Smooth page transitions to create fluid user experience.
